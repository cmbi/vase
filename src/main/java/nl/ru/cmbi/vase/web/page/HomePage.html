Welcome to the tool for Visualization of Alignments, Structure and Entropy (VASE)

VASE is a web-tool for visualizing alignments on a protein structure. It allows you to enter the ID of a protein structure in PDB format for which VASE will generate an interface. VASE will show a set of homologous sequences, aligned to the sequence of your structure, along with entropy/variability data and a 3D interface. Here's an example:

VASE can work with official PDB entries, as well as custom structures. To use your own structure, submit the contents of the PDB file in the input page and wait for the job to finish. When the alignment is done, a link to it will appear.

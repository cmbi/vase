About VASE

VASE is developed at the Centre for Molecular and Biomolecular Informatics (CMBI) department of Bioinformatics at the Radboud University in Nijmegen by C. Baakman (Coos.Baakman@radboudumc.nl) and J. Black.

Data Sources

VASE uses the structures from the Protein Databank (PDB) where available. Alignment information is taken from the stockholm files at CMBI, whenever an official PDB structure is requested. For user-submitted structures, stockholm files are generated using CMBI's webservice.

External Software

VASE uses JMOL to create a 3D representation of protein structures.

Source Code

VASE is open source and available for download at GitHub.
